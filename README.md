# Data

`odi_match_data_sample.csv` is a 20,000-row sample provided for schema inspection and
workflow testing.

The complete ODI ball-by-ball dataset is intentionally excluded because it contains more
than one million rows and may exceed GitHub's file-size limits.

To run the full analysis, place your complete dataset in this directory and update the
notebook path accordingly.

Do not upload data unless its original licence permits redistribution.
